## Установка

### Требования
- **Java 17+**
- **Windows 7> ** (для полного функционала)
  - Наличие Powershell.
- **Android** (PojavLauncher, Zalith)

### Шаги установки
1. Скачайте архив игры
3. Запустите через .bat файл

---

## Команды

### `/safe`

```
Отключает BSOD при финальном краше
```
- Отключается синий экран смерти

### `/x100`

```
Ускоряет все хоррор-эффекты в 100 раз
```

- Для быстрого тестирования
- 17 минут → ~10 секунд
